#############
biotite2pymol
#############

This package enables the usage of objects from *Biotite* directly in PyMOL,
when using its Python API:

- Import :class:`AtomArray` and :class:`AtomArrayStack` objects into PyMOL -
  without intermediate structure files.
- Convert PyMOL objects into :class:`AtomArray` and :class:`AtomArrayStack`
  objects
- Use *Biotite*'s boolean masks for atom selection as PyMOL selection
